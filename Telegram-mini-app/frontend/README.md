# Telegram Mini-App Frontend

React/Vite Telegram Mini-App for Trestle Protocol. Integrates @telegram-apps/sdk for TWA and thirdweb SDK for wallet/social login.

## Pages

- **Dashboard** — User overview
- **Marketplace** — Buy/sell digital goods
- **Tier1 Staking** — Stake Tier 1 pool
- **Tier2 Staking** — Stake Tier 2 pool
- **Tier3 Staking** — Stake Tier 3 pool
- **Withdraw** — Withdraw interface
- **TreMind AI** — AI chat assistant

## Commands

```bash
npm install
npm run dev    # http://localhost:5173
npm run build
```
