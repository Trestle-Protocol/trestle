# trestle.website

Main public-facing landing page for Trestle Protocol. Built with Next.js and thirdweb SDK.

## Sections

- **Marketing:** Hero, Features, Tokens, Roadmap, CTA
- **Dashboard:** Stake tiers, Marketplace, Wallet

## Tech Stack

- Next.js
- thirdweb SDK (wallet connection, contract interactions)
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev     # http://localhost:3000
npm run build
npm start
```
