# Reward Hub Frontend

React/Vite reward hub for Trestle Protocol. Integrates thirdweb SDK for wallet connection and TreMind AI for chat/dispute resolution.

## Pages

- **Dashboard** — User overview
- **Tasks** — Available reward tasks
- **Verify** — Biometric + source selector verification
- **Claim** — Off-chain signed reward claims
- **Leaderboard** — Weekly mindshare rankings
- **Admin** — Tasks, tiers, bonuses, verifier CRUD

## Commands

```bash
npm install
npm run dev    # http://localhost:5173
npm run build
```
